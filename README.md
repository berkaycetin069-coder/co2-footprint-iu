# CO₂-Footprint – Studienprojekt

Dieses Repository enthält den Quellcode einer Webanwendung zur Darstellung
fiktiver CO₂-Emissionsdaten. Das Projekt wurde im Rahmen einer Fallstudie
an der IU Internationale Hochschule umgesetzt.

## Technologien

- HTML5  
- CSS3  
- JavaScript (Vanilla)

## Projektbeschreibung

Die Webanwendung stellt beispielhafte CO₂-Emissionsdaten von Unternehmen
und Ländern in tabellarischer Form dar. Die Daten können gefiltert und
sortiert werden. Zusätzlich ist eine Umschaltung der Schriftkultur
(LTR/RTL) sowie eine responsive Darstellung umgesetzt.

## Hinweis

Alle dargestellten Daten sind frei erfunden und dienen ausschließlich
Studienzwecken.
