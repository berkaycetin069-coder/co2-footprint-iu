# CO₂-Footprint – Studienprojekt

Dieses Repository enthält den Quellcode einer Webanwendung zur Darstellung
fiktiver CO₂-Emissionsdaten von Unternehmen und Ländern. Das Projekt wurde
im Rahmen einer Fallstudie an der IU Internationale Hochschule entwickelt
und dient der praktischen Umsetzung grundlegender Konzepte der
Webanwendungsentwicklung.

## Projektziel

Ziel des Projekts ist es, eine übersichtliche und benutzerfreundliche
Webanwendung bereitzustellen, mit der beispielhafte CO₂-Emissionsdaten
transparent dargestellt werden können. Dabei liegt der Fokus auf einer
klaren Struktur, interaktiven Funktionen sowie einer responsiven
Darstellung auf unterschiedlichen Endgeräten.

## Funktionsumfang

Die Webanwendung bietet unter anderem folgende Funktionen:

- Darstellung fiktiver CO₂-Emissionsdaten in Tabellenform  
- Filterung der Daten nach Unternehmen oder Land  
- Sortierung der Tabelle nach verschiedenen Spalten  
- Umschaltung der Schriftkultur (LTR/RTL)  
- Responsive Darstellung für Desktop- und mobile Endgeräte  

Alle dargestellten Daten sind frei erfunden und dienen ausschließlich
Studienzwecken.

## Technische Umsetzung

Die Anwendung wurde ohne den Einsatz externer Frameworks umgesetzt.
Sämtliche Funktionen basieren auf Standard-Webtechnologien und werden
direkt im Browser ausgeführt.

### Verwendete Technologien

- HTML5
- CSS3   
- JavaScript (Vanilla) 

## Sicherheitshinweise

Nutzereingaben werden ausschließlich als Text verarbeitet. Zur Darstellung
von Inhalten wird konsequent mit sicheren DOM-Methoden gearbeitet, sodass
kein eingeschleuster Code ausgeführt werden kann.

## Hinweise zur Nutzung

Das Repository dient der Nachvollziehbarkeit der technischen Umsetzung im
Rahmen der Hausarbeit. Eine produktive Nutzung der Anwendung ist nicht
vorgesehen.

## Projektstruktur

- `index.html` – Hauptseite der Webanwendung  
- `style.css` – Gestaltung und responsives Layout  
- `script.js` – Logik für Tabelle, Filter, Sortierung und RTL-Umschaltung  
- `impressum.html` / `datenschutz.html` – rechtliche Seiten
